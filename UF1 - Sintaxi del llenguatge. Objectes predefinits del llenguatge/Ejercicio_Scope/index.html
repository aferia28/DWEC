Ejercicio 1.3.1 Scope

Dona una definició de: (sempre amb les teves paraules)

Variable Global: Una variable global es una variable que la decalramos en el "main" del programa, y de esta manera podremos utilizarla en todo el programa/aplicación
Variable Local: Una variable local es una variable que declaramos dentro de una función y solo podremos utilizarla dentro de esta función. Si la utilizaramos fuera de la función el programa no la reconocería.
Ámbito: Es donde se delimita el uso de una variable. Por ejemplo una variable local de una funcion tendría el ambito de uso en la parte del codigo entre los "{}"
Scope: Es en que ambito se puede utilizar una variable.
