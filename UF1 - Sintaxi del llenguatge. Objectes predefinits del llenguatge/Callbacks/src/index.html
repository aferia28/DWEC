Funcions

La versió clàssica()farà una execució seqüencial de les funcions. El programa anirà informant a la consola d'on està i que està fent.

La versió ambCallback(funcioAexecutar) serà una funció que recull com a paràmetre una altra funció. De manera aleatòria executarà la funció tercera o la funció segona.

El treball d'aquesta funció assíncrona serà llençar un alert i escriure alguna cosa a la consola 5 segons desprès d'haver acabat l'execució de la funció que la va crear.

function clasica(){
    console.log("Empieza la ejecucion de clasica()\n Llamo a segunda() desde clasica()");
    segona();
    console.log("Acabo de llamar a segona() desde clasica();");     console.log("Llamo a tercera() desde clasica()");     tercera();
    console.log("Acabo de llamar a tercera() desde clasica()");     console.log("Acaba la ejecucion de clasica()");
  }

function segona(){
    console.log("-------------------------------");     console.log("Empieza la ejecucion de segunda()\n Entro en el bucle llarg()\n Surto del bucle llarg()\n Acaba la ejecucion de segunda()\n");
    console.log("-------------------------------");
  }

function tercera(){
    console.log("-------------------------------");     console.log("Empieza la ejecucuon de tercera()\nEntro al bucle llarg()\n Surto del bucle llarg() \n Acaba la ejecucion de tercea()\n");
    console.log("-------------------------------");
  }

function ambCallback(funcioAexecutar){
    console.log("-------------------------------");     console.log("Empieza la ejecucion cambCallback");     funcioAexecutar();
    console.log("Termina la funcon ambCallback()")     console.log("-------------------------------");
}

function assincrona(funcioAexecutar){
    funcioAexecutar();
  }
